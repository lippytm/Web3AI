# Web3AI Frontend

Next.js frontend for the Web3AI application.

## Setup

Install dependencies:
```bash
npm install
```

Configure environment:
```bash
cp .env.example .env.local
# Edit .env.local with your configuration
```

## Development

Run development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Building

Build for production:
```bash
npm run build
npm start
```

## Linting & Formatting

Run ESLint:
```bash
npm run lint
```

Format code:
```bash
npm run format
```

Check formatting:
```bash
npm run format:check
```

Type check:
```bash
npm run type-check
```

## Testing

Run tests:
```bash
npm test
```
